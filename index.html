HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Elements that are nested inside other elements are called child elements.
  • The element that holds all the children is known as the parent element.
  • When working with visible HTML elements, the content is contained between an opening tag and a closing tag.

CSS

  • A selector defines the element or attributes to which the rules, or declarations, will apply.
  • Declarations contain two important components: the CSS property we want to apply and the value of the property.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class selector.
  • Any HTML element assigned to the class selector, card, will gain these property assignments from the CSS.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: switches to main branch
  • git pull origin main: pulls down the latest main branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • The console.log method was created to make it easier for developers to test their code.
  • Control flow is the order in which a computer executes code in a script.
  • We can use conditional statements and loops to control the order in which code is executed, or if it is even executed at all.
  • Using conditional statements such as "if" statements allows us to interrupt control flow.
  • If one condition is truthy, the computer will execute the code that we provide between curly brackets.
  • But if the condition is not truthy, the code will not execute, interrupting the control flow.
  • An array is a single variable that is used to hold a group of data.
  • The location of the data item in the array is identified with a unique number called an index.
  • A "for" loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
  • A "for" loop contains the following three important statements: starting point for our loop, condition, and what allows the array to iterate over each item.
  • To comment out multiple lines of JavaScript, put /* before and */ after the block of code.
  • At the most basic level, a function is simply a set of instructions that tells the computer how to perform a certain task.
  • When we create a function, we are defining a block of code that performs a specific task and giving it a unique name.
  • You must call a function by name in order to execute it.
